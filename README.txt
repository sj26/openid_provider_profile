OPENID PROVIDER PROFILE
=======================

The module uses the schema definitions from http://www.axschema.org/types/

INSTALLATION
------------

Install and enable the OpenID Provider Proile Drupal module as you would any Drupal module.

DEPENDENCIES
------------

Module relies on the OpenID Provider Module and OpenID Provider AX Module.

Compatible with Drupal 6.x
